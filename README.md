## Comando para utilizar a cli do nest
```bash
$ npx @nestjs/cli g <OPTIONS>
```

## Info
- Node version: 18.13.0
- NPM Version: 9.2.0
- CLI Version: 9.2.0

## Rodando o projeto
- $ docker-compose up -d

# Acessando o mongo localmente
```bash
$ http://localhost:8081
```









## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
